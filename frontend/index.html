About

This project aims to develop an AI/ML model to generate fine spatial resolution air quality maps from coarse resolution satellite data.

Methodology

The methodology involves data collection, preprocessing, feature engineering, model selection, training, validation, and post-processing.

Results

The model's performance is evaluated using metrics such as MAE, RMSE, and R-squared.
